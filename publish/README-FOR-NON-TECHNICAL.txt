========================================
   SUPPORT COPILOT AGENT - QUICK START
========================================

FOR NON-TECHNICAL TEAM MEMBERS:

1. EXTRACT THIS FOLDER to your desktop or any location

2. DOUBLE-CLICK "START-APP.bat"
   - A command window will open
   - Your browser will open automatically
   - The app will start at: http://localhost:5097

3. USE THE APP:
   - Enter a JIRA incident ID (e.g., CMSRC-8930)
   - Click "Analyze Incident"
   - See AI-powered resolution suggestions!

4. TO STOP:
   - Close the command window, OR
   - Press Ctrl+C in the command window

========================================
   TROUBLESHOOTING
========================================

Problem: "Can't reach the page"
Solution: Wait 10-15 seconds after starting, then refresh browser

Problem: Command window closes immediately
Solution: Make sure you extracted all files, not just the .bat file

Problem: Port already in use
Solution: Close any other applications using port 5097
